## 💳 דרכי התשלום שאנחנו נוסיף

### 1. Stripe - כרטיסי אשראי בינלאומיים ⭐ **חובה**

**מתי לפי התוכנית:** ימים 10-11

**מה זה:**
- כרטיסי אשראי (Visa, Mastercard, American Express)
- תשלומים בינלאומיים
- תמיכה בשקלים, דולרים, אירו

**למי זה מיועד:**
- ✅ לקוחות מישראל (עם כרטיסי אשראי)
- ✅ לקוחות מחו"ל (כל העולם)
- ✅ כל מי שיש לו כרטיס אשראי

**איך זה יעבוד:**
1. הלקוח בוחר שירות ומזמין
2. המערכת מעבירה אותו לעמוד תשלום Stripe
3. הלקוח מזין פרטי כרטיס אשראי
4. Stripe מעבד את התשלום
5. הכסף מתווסף ליתרה של המשתמש
6. המשתמש יכול ליצור הזמנות

**עלויות:**
- תשלום חינמי לך (אתה לא משלם ל-Stripe על ההתקנה)
- תשלום רק כשמשלמים לך: ~2.9% + ₪0.90 לכל תשלום
- לדוגמה: תשלום של ₪100 → תשלם ל-Stripe ~₪3.80, תקבל ₪96.20

**מה צריך:**
- רישום ב-stripe.com (חינמי)
- API Keys (Publishable + Secret)
- **מתי:** לפני יום 10, או אחרי שנסגור פרטים בימים 1-9

### 2. bit - תשלומים ישראליים 🇮🇱 **אופציונלי**

**מתי לפי התוכנית:** ימים 12-13

**מה זה:**
- bit, שח"ח (העברה בנקאית)
- כרטיסי אשראי ישראליים
- תשלומים בשקלים בלבד

**למי זה מיועד:**
- ✅ לקוחות מישראל בלבד
- ✅ מי שמעדיף תשלום דרך bit/העברה בנקאית
- ✅ מי שלא רוצה להשתמש בכרטיס אשראי

**איך זה יעבוד:**
1. הלקוח בוחר שירות ומזמין
2. המערכת מעבירה אותו לעמוד תשלום bit
3. הלקוח בוחר דרך תשלום (bit/שח"ח/כרטיס)
4. bit מעבד את התשלום
5. הכסף מתווסף ליתרה
6. המשתמש יכול ליצור הזמנות

**עלויות:**
- תלוי ב-bit - בדרך כלל ~1.5-2% + עמלה קטנה
- זול יותר מ-Stripe לתשלומים ישראלים

**מה צריך:**
- רישום עסקי ב-bit.co.il (אם נדרש)
- API Key מ-bit
- **מתי:** לפני יום 12, או אחרי שנסגור פרטים
